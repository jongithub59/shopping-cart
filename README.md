# Shopping Cart

## Description

A shopping cart application where you can view the items of Deadlock and purchase them to cresate your own build. From the Odin Project [curriculum](https://www.theodinproject.com/lessons/node-path-react-new-shopping-cart)

Live preview [here](https://jongithub59.github.io/shopping-cart). 

## Features:

* Loads and display the items based on the category selected
* Hover on any item to view its stats and description
* Click to buy an item (items can only be purchased once)
* View your items in cart and discard any unwanted items

## Reflections:

* Learned about fetching a large quantity of data and how to efficiently sort it for easy rendering
* Learned to utilize routes to create differnt pages for React applications
* Learned how to worrk with an API to get all the data necessary and nothing else
